# Persona HRIS

Sistem Informasi Sumber Daya Manusia (HRIS) lengkap dengan fitur manajemen karyawan, absensi, cuti, dan payroll.

## 🚀 Fitur Utama
- **Manajemen Karyawan**: CRUD, export Excel/PDF
- **Absensi**: Check in/out manual + QR Code
- **Cuti**: Pengajuan, approval, sisa cuti otomatis
- **Payroll**: Generate otomatis, slip gaji PDF
- **Profile**: Data pribadi, upload foto
- **Notifikasi Email**: Status cuti via email

## 🛠️ Tech Stack
- **Frontend**: Next.js 15, TypeScript, Tailwind CSS
- **Backend**: Supabase (PostgreSQL, Auth, Storage)
- **Testing**: Playwright
- **Deployment**: Vercel
